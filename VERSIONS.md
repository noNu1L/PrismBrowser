# Versions

This file tracks the versions of key third-party components used in PrismBrowser.

## Core Components

- **Electron**: `^29.1.5`
  - The framework used to build the desktop application.
  - Source: `package.json`

- **Clash.Meta (Mihomo)**: `v1.19.10`
  - The core proxy engine.
  - Source: `clash-meta/mihomo.exe -v`

- **Yacd-meta (Dashboard)**: `v0.3.7`
  - The web-based management UI for Clash.
  - Source: Corresponds to the latest release version of the `gh-pages` branch from [MetaCubeX/Yacd-meta](https://github.com/MetaCubeX/Yacd-meta).
