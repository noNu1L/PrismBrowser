# Versions(版本信息)

This file tracks the versions of key third-party components used in PrismBrowser.
本文件用于记录 PrismBrowser 项目中所用关键第三方组件的版本。

## Core Components(核心组件)

- **Electron**: `^29.1.5`
  - The framework used to build the desktop application.(构建桌面应用的主框架)
  - Source: `package.json`

- **Clash.Meta (Mihomo)**: `v1.19.10`
  - The core proxy engine.(代理核心引擎)
  - Source: `clash-meta/mihomo.exe -v`

- **Yacd-meta (Dashboard)**: `v0.3.7`
  - The web-based management UI for Clash.(Clash 的 Web 管理界面)
  - Source: Corresponds to the latest release version of the `gh-pages` branch from [MetaCubeX/Yacd-meta](https://github.com/MetaCubeX/Yacd-meta). 
  - 来源：对应 [MetaCubeX/Yacd-meta](https://github.com/MetaCubeX/Yacd-meta) 的 `gh-pages` 分支最新发布版本。
